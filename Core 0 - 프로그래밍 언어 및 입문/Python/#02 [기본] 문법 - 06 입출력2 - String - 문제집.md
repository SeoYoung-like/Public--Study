# 입출력2 - String과 문자열 포맷팅

* **컨테이너는 무엇인가? **123
  * 여러 데이터를 담아놓은 자료형이다.



* **시퀀스는 무엇인가? **123
  * 컨테이너형  중에서 원소(element)들이 순서대로 나열되어 있는 것을 말한다.



* **시퀀스 자료형에는 어떤 것이 있는가? (3가지)** 123
  * 문자열, 리스트, 튜플이 있다.



* **시퀀스의 특징은 무엇인가? **123

  |

  * 인덱싱과 슬라이싱을 사용할 수 있다.



* **작은 따옴표 큰 따움표 출력하는 방법 (4가지) **123
  * 작은따옴표 사용은 큰따옴표로 묶기
  * 큰따옴표 사용은 작은따옴표로 묶기
  * 여러 줄 문자열로 묶기 - `""" """`, `''' '''`
    * 작은 따옴표, 큰 따옴표를 여러 개 출력해야 할 때 사용한다.
  * 이스케이프 문자 사용



* **[참고] 작은 따옴표 큰 따움표 출력하는 코드 ** ㅇㅇㅇ

  * ```python
    >>> s = "Python isn't difficult"
    >>> s
    "Python isn't difficult"
    ```

  * ```py
    >>> s = 'He said "Python is easy"'
    >>> s
    'He said "Python is easy"'
    ```

  * ```python
    single_quote = '''"안녕하세요."
    '파이썬'입니다.'''
     
    double_quote1 = """"Hello"
    'Python'"""
     
    double_quote2 = """Hello, 'Python'"""    # 한 줄로 작성
     
    print(single_quote)
    print(double_quote1)
    print(double_quote2)
    ```

    ```

  * ```python
    >>> s = "He said \"Python is easy\""

* **슬라이싱 구조를 설명하시오.** 123
  * [시작 번호 : 끝 번호 : 스텝(생략 가능)]
    * 끝 번호에 해당하는 것은 포함하지 않는다.
    * 음수를 사용할 수도 있다.



* **[참고] 스텝을 사용하는 슬라이싱을 확장 슬라이싱 이라고 불리기도 한다.**  ㅇㅇㅇ
  * extended Slices



* **슬라이싱의 시작 번호나 끝 번호를 생략하면 어떻게 되는가?** 123
  * 맨 처음 이나 맨 끝으로 지정된다.

* **문자열의 특정 문자를 인덱싱을 사용해서 바꿀 수 있는가?** 123
  * 불가능하다. ( 문자열의 불변성 )
  * 문자열은 immutable한 자료형이기 때문에 요솟값을 수정 할 수 없다.
    * 대신, 수정된 객체를 새로 만들어야 한다.



* **[참고] 문자열 새로운 객체 생성** ㅇㅇㅇ

  * ```python
    # 슬라이싱으로 새로운 객체 생성
    b = a[1:]
    print(id(a))
